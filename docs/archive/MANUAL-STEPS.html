  1. Click "Create"

3
Add Environment Variables to Vercel
You'll need: The values from setup.sh output + the Google OAuth values you just created

Copy and run these commands:

printf 'STAGING_DB_URL' | vercel env add DATABASE_URL preview
printf 'PRODUCTION_DB_URL' | vercel env add DATABASE_URL production
printf 'NEXTAUTH_SECRET' | vercel env add NEXTAUTH_SECRET preview production
printf 'https://STAGING_URL' | vercel env add NEXTAUTH_URL preview
printf 'https://PRODUCTION_URL' | vercel env add NEXTAUTH_URL production
printf 'GOOGLE_CLIENT_ID' | vercel env add GOOGLE_CLIENT_ID preview production
printf 'GOOGLE_CLIENT_SECRET' | vercel env add GOOGLE_CLIENT_SECRET preview production

Redeploy to apply changes:

git commit --allow-empty -m "Trigger redeploy" && git push
